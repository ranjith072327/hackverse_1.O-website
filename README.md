# Hackverse 1.0 Registration Website

This project is a custom hackathon registration website built using:

- HTML
- CSS
- JavaScript
- Google Apps Script
- Google Sheets
- Google Drive

## Project Purpose

This website is designed for **Hackverse 1.0 – 24-Hour Hackathon Registration**.

It allows participants to:

- enter team details
- enter member details
- upload payment screenshot
- submit registration online

The submitted data is stored in:

- **Google Sheets** for registration records
- **Google Drive** for uploaded payment screenshots
